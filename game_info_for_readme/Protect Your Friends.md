## Features

### 1. Breakables
- Break Wood with **Golden Axe**
- Break Stone with **Golden Pickaxe**
- **Auto-Break** toggle for continuous breaking (stone is broken for autobreak)
- **Refetch Breakables** button to update target list (u probs wont have to)

### 2. Campfire Contributions
- Contribute **Logs** (100) (also removes the limit xd so u can have 14k wood/100)
- Contribute **Stone** (100) (also removes the limit xd so u can have 10k stones/50. U cant have both tho)
- **Auto-Contribute** toggle for Logs and Stone

### 3. Bridge Contributions
- Contribute **Logs** (10)
- Contribute **Stone** (10)
- **Auto-Contribute** toggle for Logs and Stone

### 4. Use Potion
- Use **Logs** potion
- Use **Stone** potion
- **Auto Use Potion** toggle

### 5. Potion Shop
- Grab **Logs** potion
- Grab **Stone** potion
- **Auto Grab Potion** toggle
